# Enumerable-methods

![](https://img.shields.io/badge/Microverse-blueviolet)

Replication of the Enumerable methods (each, each_with_index, map, inject, select, all, any, none, count) in Ruby with the use of yield and procs.

## Built With

- Ruby

## Getting Started

To get a local copy up and running follow these simple example steps.

- `git clone https://github.com/ourandco/Enumerable-methods`
- `cd Enumerable-methods`
